{% trans "Las Calas que más nos gustan de Menorca" %}

{% blocktrans %} En Menorca hay que andar para llegar a las calas, en algunos casos mucho y los caminos pueden ser bastante complicados. Usa calzado cómodo para las caminatas (calzado de trekking, deportivos, escarpines...etc) {% endblocktrans %}

{% blocktrans %} Los parkings cercanos a algunas calas se llenan enseguida, por lo que hay que intentar llegar pronto. A partir de las 10.00 - 11.00 en pleno verano los parking están completos, quizá tengamos algo más de suerte en Septiembre. {% endblocktrans %}

{% blocktrans %} Las calas escondidas y playas vírgenes no tienen ningún tipo de servicio (como debe ser!) más vale que lleves comida y bebida contigo si quieres pasar el dia. {% endblocktrans %}



{% blocktrans %} 1. Binidali, {% endblocktrans %}

{% blocktrans %} Sin duda nuestra cala! Muy cerquita de Benibeca. {% endblocktrans %}



{% blocktrans %} 2. Pregonda (Norte) {% endblocktrans %}

{% blocktrans %} Es una playa especial para nosotros, es super bonita y tranquila. {% endblocktrans %}

{% blocktrans %} Lleva comida y bebida para pasar el dia. {% endblocktrans %}



{% blocktrans %} 3. Cala Escorxada (Sur) {% endblocktrans %}

{% blocktrans %} Es la gran desconocida del sur, muy poca gente la visita sobre todo porque hay que andar al menos una hora para llegar, eso sí las vistas durante el camino son inigualables. {% endblocktrans %}

{% blocktrans %} Nos la recomendó una chica de Menorca, es la favorita de muchos menorquines. {% endblocktrans %}

{% blocktrans %} Es un destino frecuente de barcos que fondean cuando sopla viento norte. No hay servicios de ningún tipo. {% endblocktrans %}



{% blocktrans %} 4. Es Talaier (Sur) {% endblocktrans %}

{% blocktrans %} Es el prototipo de playa del sur de Menorca. Arena fina y agua turquesa! Es como una piscina natural! {% endblocktrans %}

{% blocktrans %} Para llegar tienes que hacer excursión desde Son Saura, la playa es bastante pequeña y en plena temporada de verano suele estar llena… aún así, merece la pena una visita. {% endblocktrans %}



{% blocktrans %} 5. Cavalleria (Norte). {% endblocktrans %}

{% blocktrans %} Una playa preciosa de arena roja. {% endblocktrans %}

{% blocktrans %} Es muy visitada por varias razones, se encuentra en un espacio natural único, está en el paso de Camí de Cavalls y tiene un parking a solo 5 minutos de la playa. {% endblocktrans %}

{% blocktrans %} Tambien podeis visitar el Faro de Cavalleria que está muy cerca y las vistas son super bonitas. {% endblocktrans %}



{% blocktrans %} 6. Cala en Turqueta {% endblocktrans %}

{% blocktrans %} Si tuviéramos que elegir una sola cala.. No podríamos! Pero Cala en Turqueta seguro que sería una de las candidatas! {% endblocktrans %}

{% blocktrans %} La hemos visitado mucho desde nuestros primeros viajes a Menorca. {% endblocktrans %}



{% blocktrans %} 7. Cala Mitjana (y Mitjaneta) {% endblocktrans %}

{% blocktrans %} Playa de aguas turquesas y transparentes, rodeada de pinos. {% endblocktrans %}



{% blocktrans %} 8. Cala Trebaluger {% endblocktrans %}

{% blocktrans %} Cala virgen que está a 40 minutos siguiendo un camino que se toma desde la izquierda desde Mitjana, con la ayuda de unos escalones de piedra. Hay que subir una fuerte pendiente hasta llegara lo alto del acantilado. {% endblocktrans %}

{% blocktrans %} Desde alli un camino plano, dominado por un bosque de pinos te llevará hasta cala Trebaluger. {% endblocktrans %}

{% blocktrans %} Para acceder a la playa deberás bajar por otro tramo con bastante desnivel y más escalones de piedra. {% endblocktrans %}

{% blocktrans %} Si continuas andando desde Cala Trebaluger, llegamos a nuestra preferida número 3 Escorxada! {% endblocktrans %}

{% blocktrans %} Por supuesto hay muchas otras, igual de bonitas pero queríamos compartir nuestras favoritas con vosotros. {% endblocktrans %}



{% blocktrans %} Otras: {% endblocktrans %}

{% blocktrans %} Las famosas Macarella y Macarelleta, sin duda las playas más populares de Menorca. {% endblocktrans %}

{% blocktrans %} Otras que aún no conocemos, y eso nos encanta, que a pesar del número de veces que hemos visitado la isla aún queda mucho por descubrir! Cala Presili, Cala Pilar, Binigaus Y Santo Tomás. {% endblocktrans %}

{% blocktrans %} Algunas de fácil acceso: Pla de Son Bou, playa de Cala en Porter, playa de Bininequer, Cala Punta Prima {% endblocktrans %}
